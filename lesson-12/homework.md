# Домашнее задание к лекции №12

### Установить и поднять centrifugo, прикрутить к проекту, продемонстрировать отправку какого-то сообщения с помощью websocket

Нужно создать html-страницу для отображения всех продуктов (фильмы, музыка, книги, игры, схемы вязания и т.д.). Далее прикрутить centrifugo.

- Устанавливаем https://github.com/centrifugal/centrifugo/blob/master/docs/content/server/install.md
- Генерируем конфиг
```
centrifugo genconfig
```
- В настройках бекенда регистрируем Centrifugo secret and Centrifugo API
key (дефолтный адрес http://localhost:8000/api)
- Подключаем библиотеку для клиента
https://github.com/centrifugal/centrifuge-js
- Подписываем клиента на каналы

При демонстрации нужно показывать html-страницу со списками всех продуктов. А в фоновом режиме (например через консоль) сделать POST-запрос добавления нового продукта, чтобы на html-странице он появился без перезагрузки.
